## 规范说明

### PDF 格式
- Logo: finding_technology_logo_v12.png
- Logo 尺寸：高 1.5cm, 宽 5.4cm
- 页眉：Logo 右对齐，底部灰线
- 页脚："make life easy" + 页码
- 页面：A4, 上 3.5cm, 下 2cm, 左右 2cm

### 五看三定结构
1. 看产业 - 价值链、趋势、赛道选择、PESTEL
2. 看市场 - 3 赛道×9 细分市场 (TAM/AM/TM, VOC+KANO, SWOT)
3. 看自己 - 公司简介、优劣势
4. 战略总结 - 市场汇总、3-5 年规划
